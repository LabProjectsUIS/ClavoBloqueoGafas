Detailed Description

Implements live retargeting of streamed OptiTrack skeletal data via Mecanim. Add this component to an imported model GameObject that has a properly configured humanoid Avatar definition.

A hierarchy of GameObjects (see m_rootObject and m_boneObjectMap) will be created to receive the streaming pose data for the skeleton asset specified by SkeletonAssetName. A source Avatar will be created automatically from the streamed skeleton definition.
