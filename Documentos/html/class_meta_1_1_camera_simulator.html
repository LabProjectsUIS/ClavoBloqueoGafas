Detailed Description

This script is used to emulate calibrated cameras in Unity. It uses camera intrinsics to create an "opengl-style" projection matrix for a rendering camera.

*** Attach this script to a gameobject that has a camera component.

See equation used for projection matrix below:

[2*K00/width, -2*K01/width, (width - 2*K02 + 2*x0)/width, 0] [ 0, -2*K11/height, (height - 2*K12 + 2*y0)/height, 0] [ 0, 0, (-zfar - znear)/(zfar - znear), -2*zfar*znear/(zfar - znear)] [ 0, 0, -1, 0]
