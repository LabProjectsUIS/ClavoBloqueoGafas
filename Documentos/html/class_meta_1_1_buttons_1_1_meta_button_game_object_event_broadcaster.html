Detailed Description

Broadcast the button events from the headset via GameObject.FindObjectsOfType. This class is not performant but is the easiest way to interact with buttons. For more performant implementations please see MetaButtonIndividualEventBroadcaster, MetaButtonGeneralEventBroadcaster or MetaButtonEventBroadcaster.
